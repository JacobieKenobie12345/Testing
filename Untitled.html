    <div id = "what-are-shadows" ><h2 >What are <span class = " Shadows">shadows</span>?</h2>

    <p>A <span class = "Shadows">shadow</span> is simply just a place where there is an object in front of a light source. For example if I am camping and I need to go to the bathroom in the middle of the night. I take a flashlight to see where I am going and, all the trees will have <span class = "Shadows">shadows</span> because they are blocking the light. Some of you are probobly wondering why aren't <span class = " Shadows">shadows</span> all of the way black? Well the answer to that question is that light bounces off other things and comes to where the <span class = "Shadows">shadow</span> is. </p></div>
